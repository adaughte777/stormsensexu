A small Streamlit app that estimates evacuation risk and returns a packing list based on forecast and household details.

How to run locally

```
pip install -r requirements.txt
streamlit run app.py
```

Deploy to Streamlit Cloud

1. Push this repository to GitHub (public).
2. Go to https://share.streamlit.io and connect your GitHub account.
3. Select the repo and branch and set the app file to `app.py`.

Sample URL

https://share.streamlit.io/adaughte777/stormsensexu/main/app.py
